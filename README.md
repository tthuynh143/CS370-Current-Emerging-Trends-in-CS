# CS370-Current-Emerging-Trends-in-CS

     In this project, I developed an autonomous agent designed to navigate a maze to locate treasure using reinforcement learning. The foundation of the project was built upon a provided Python framework that included the environment dynamics, the maze visualization tools, and the basic structure for the experience replay memory. My primary contribution involved implementing the Deep Q-Network (DQN) logic and the training loop. I created the neural network architecture using Keras, defined the loss functions, and scripted the epsilon-greedy strategy to balance exploration and exploitation. By fine-tuning hyperparameters such as the discount factor and learning rate, I was able to transform a random-moving agent into an intelligent system that successfully optimized its path to the treasure within just a few training epochs.

     As a computer scientist, I see my role as a professional problem solver who utilizes computational tools to create efficiency and solve complex logic puzzles. This work matters because it extends beyond simple games; the same reinforcement learning principles used in this maze can be applied to logistics, autonomous vehicles, and adaptive resource management. We act as the architects of the digital world, translating messy, real-world challenges into structured algorithms that can be tested, refined, and scaled. By building these systems, we provide the infrastructure for modern innovation, making data-driven decision-making faster and more accurate than humanly possible.

It is a cycle of abstraction and implementation that ensures the final solution is not just functional, but also robust and maintainable.
